# 공유오피스 이용자 결제 전환 예측 및 회원권 최적화 프로젝트

## 📌 Overview
무료 체험 이용자의 **결제 전환율 하락(최근 30% 이하)** 문제를 해결하기 위해  
예측 모델을 개발하고, 행동 데이터 기반으로 **회원권·프로모션 개선 방안**을 제안한 프로젝트입니다.

---

## 🎯 Objectives
- 무료 체험 이용자의 **결제 전환 여부 예측**
- 전환에 영향을 주는 **핵심 요인 분석**
- 이용 패턴 기반 **회원권 구조 재설계**
- 방문 패턴을 고려한 **무료 체험 제도 개선**

---

## 🧩 Data
- 내부 로그: 방문 시간대, 방문 횟수, 총 이용시간, 마지막 방문 등  
- 외부 데이터: 기상청 날씨, 인구수, 상권, 지점 정보  
- 수도권 9개 지점 기준 데이터 결합

---

## 🛠 Modeling
여러 모델 비교 후 **Gradient Boosting**을 최종 선정  
- **AUC: 0.658 (Best)**  
- 주요 변수:  
  - 일 평균 이용시간  
  - 방문 일수  
  - 방문 시간대  
  - 총 이용시간  
  - 마지막 방문 시점  

➡ 결제 전환은 **이용 패턴 변수(약 80%)**가 가장 강하게 영향을 미침

## 📈 Key Insights
- **이용 시간이 길수록 결제율 감소** (180분 이상부터 유의하게 하락)  
- **주간 이용자**는 오래 이용하지만 결제율은 낮음  
- **무료 체험 방문 일수·간격이 길수록** 결제율 상승
  
---

## 🚀 Recommendations

### 1) 회원권 최적화
| 항목 | 개선안 |
|------|--------|
| 무제한 권 | 385,000원 → **270,000원** |
| 신규 정액권 | **50시간권 99,000원 / 100시간권 179,000원** |
| 기본권 및 야간/주말권 | 현행 유지 |

➡ 실제 이용 패턴을 반영한 가격·시간 구조

### 2) 무료 체험 제도 개선
- 기존: 3일 연속 이용  
- 개선: **7일 기간 중 원하는 3일 선택**  
→ 방문 분산, 만족도 증가, 전환율 개선 기대

---

## ⚠ Limitations
- 인구통계 데이터 부재  
- 무료 체험 이용자 중심 분석  
- 마케팅 이벤트 영향 완전 제거 어려움
