# 구독 기반 교육 콘텐츠 서비스 회원 행동 분석

## 📌 Overview
이 프로젝트는 **구독 기반 교육 콘텐츠 서비스의 첫 구독 전환율 및 재구독율을 향상**시키기 위해  
회원 행동 데이터를 분석하고, 데이터 기반 전략을 제시하는 것을 목표로 한다.

---

## 🎯 Objectives
- 회원의 **첫 구독 전환 과정** 분석
- **재구독 행동 패턴** 파악
- **RFM 기반 세그먼트 분류**
- 구독 확대를 위한 전략 도출

---

## 📁 Data & Preprocessing
사용 데이터:
- 가입 정보(`complete_signup`)
- 결제 데이터(`paid`)
- 콘텐츠 이용 기록(`content_merged`)
- 무료 체험 데이터(`start_free_trial`)

---

## 📊 Key Analyses

### 1) 퍼널 분석 (첫 구독 전환)
- 가입 → 레슨 시작: **64.3%**
- 레슨 → 결제 페이지: **~40%**
- 결제 페이지 → 결제 완료: **24%**  
➡️ **결제 페이지 개선이 최우선 과제**

### 2) 무료 체험 효과성
- 체험자 전환율: **42.4%**
- 비체험자 전환율: **15.0%**
- p-value < 0.001 → **무료 체험 효과 유의미**

### 3) 재구독 패턴
- 1회 구독: 평균 **3.72개**, 완료율 **25%**
- 2회 이상 구독: 평균 **5.90개**, 완료율 **36%**
- 재구독자는 중급·고급 콘텐츠 비중 증가

### 4) RFM 세그먼트 요약
- A(충성): 8% — 활동·결제 가장 활발
- B(우수): 17%
- C(이탈 위험): 27%
- D(비활성): 47% — 장기 미이용자 비중 높음

---

## 🎯 Key Insights
- **무료 체험은 전환율 증가에 큰 영향**  
- **콘텐츠 페이지 4회 방문** 시 무료 체험 유도 효과가 가장 높음  
- **초급 콘텐츠 완강률 개선**이 재구독율 향상에 중요  
- 고급 콘텐츠의 **레슨 수 과다**가 완료율 저해 가능  

---

## 🚀 Recommendations

### Short-term
- 결제 페이지 UI/UX 개선  
- 콘텐츠 4회 방문자 대상 무료 체험 리마인드  
- 초급 콘텐츠 강화 및 고급 난이도 조정  

### Long-term
- 인구통계 데이터 확보  
- 세부 콘텐츠 카테고리 수집  
- 생존 분석·추천 모델 도입  
- 세그먼트별 개인화 전략 확대
